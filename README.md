# GTchat para Contabilidades

Landing page estática, responsiva e sem dependências.

## Arquivos

- `index.html`: estrutura importável.
- `style.css`: estilos isolados.
- `script.js`: menu, abas, FAQ e formulário.
- `design-reference.png`: referência visual gerada.

## WordPress

Opções de importação:

1. Tema filho: copiar HTML para template PHP e enfileirar CSS/JS.
2. Elementor: usar widget HTML e carregar CSS/JS globalmente.
3. Plugin de snippets: inserir corpo HTML, CSS e JavaScript separadamente.

Troque envio simulado do formulário em `script.js` pela URL do CRM, webhook ou plugin WordPress.
